# 内容安全指南

本指南确保所有分镜内容符合安全标准，避免暴力、血腥或其他不适描述。

---

## 禁止内容

### 1. 暴力相关
- **身体伤害**：击打、流血、伤口、痛苦表情
- **武器使用**：枪械、刀具、打斗动作
- **威胁场景**：追逐、恐吓、强迫

### 2. 血腥相关
- **血液**：任何血液或流血场景
- **伤口**：开放性伤口、疤痕、医疗创伤
- **死亡**：尸体、丧葬、临终场景

---

## 替换策略

当检测到暴力或血腥描述时，使用以下替代方案保持镜头效果：

### 身体冲突 → 转化为抽象表达

| 原描述 | 替换方案 | 保持效果 |
|--------|---------|---------|
| 角色被打倒 | 角色快速后退/身体倾斜失衡 | 戏剧张力 |
| 流血 | 衣物破损、能量消散粒子 | 损伤感 |
| 痛苦表情 | 面部模糊、剪影轮廓 | 情绪传达 |
| 武器指向 | 手势引导、能量光束 | 紧张感 |
| 追逐打斗 | 光影追逐、符号对抗 | 冲突感 |

### 紧张/危机场景 → 转化为视觉隐喻

| 原描述 | 替换方案 | 保持效果 |
|--------|---------|---------|
| 角色受伤倒地 | 角色坐于破碎地面、周围物体散落 | 脆弱感 |
| 生死抉择 | 分岔路口、天平倾斜 | 决策压力 |
| 危险环境 | 昏暗光线、狭窄空间、不稳定结构 | 危机感 |
| 激烈对抗 | 色彩对比、光影交错、元素碰撞 | 冲突张力 |

### 情绪高潮 → 使用符号化表达

| 情绪 | 符号化方案 | 视觉元素 |
|------|-----------|---------|
| 愤怒 | 红色色调、尖锐几何形状 | 火焰光效、能量波动 |
| 悲伤 | 蓝色色调、下雨效果 | 雨滴、倒影、缓慢下落 |
| 恐惧 | 阴影、模糊边缘 | 扭曲空间、不稳定构图 |
| 绝望 | 黑暗、孤立元素 | 空旷场景、小型主体 |

## 运镜替代方案

保持戏剧效果的同时避免暴力表达：

### 紧张镜头
- **快速切换**：使用快速剪辑而非真实打斗
- **特写强调**：手部颤抖而非伤口特写
- **阴影表现**：使用阴影和剪影暗示而非直接展示
- **音效强化**：心跳声、呼吸声替代击打声

### 危机镜头
- **环境暗示**：破碎玻璃、倒塌物品而非身体伤害
- **象征元素**：断裂的链条、熄灭的火焰
- **构图张力**：倾斜构图、不稳定视角
- **光影对比**：强明暗对比营造紧张感

### 冲突镜头
- **色彩对抗**：冷暖色调对比替代肢体冲突
- **元素碰撞**：光与暗、水与火等自然元素
- **空间压迫**：墙壁逼近、空间缩小
- **时间凝滞**：慢动作强调关键时刻

## 角色表达安全规范

### 面部表情
- ✅ 使用：眼神方向、轮廓清晰度、光影角度
- ❌ 避免：痛苦的扭曲、狰狞表情、流血

### 身体姿态
- ✅ 使用：开放/闭合姿态、高度变化、空间位置
- ❌ 避免：受伤的佝偻、战斗姿势、倒地姿势

### 环境互动
- ✅ 使用：物体位移、光线变化、粒子效果
- ❌ 避免：破坏行为、攻击性动作

---

## 审核清单

在生成分镜前检查：

- [ ] 是否有身体伤害描述？
- [ ] 是否有血液或伤口？
- [ ] 是否有武器或攻击动作？
- [ ] 是否有死亡或临终场景？
- [ ] 是否有过度恐怖的描述？
- [ ] 是否可以使用隐喻替代？
- [ ] 是否保持了戏剧张力？

---

## 最佳实践

1. **抽象化表达**：将具体伤害转化为抽象视觉元素
2. **符号化语言**：使用普遍认可的视觉符号传达情绪
3. **环境叙事**：通过环境变化暗示事件而非直接展示
4. **音效补偿**：使用音效和音乐强化情绪替代视觉冲击
5. **留白艺术**：给观众想象空间，避免过度描述

## 示例转换

### 示例1：动作场景

**原描述（不安全）：**
```
角色被击中胸口，吐血倒地，表情痛苦
```

**安全替换：**
```
角色身体后仰失衡，衣物飘扬，周围物体散落，能量粒子从胸口位置向外扩散，地面出现裂纹，角色缓慢坐下
```

### 示例2：危机场景

**原描述（不安全）：**
```
刀架在角色脖子上，角色脸色苍白
```

**安全替换：**
```
角色脖颈处出现光影分界线，冷色调笼罩面部，背景光逐渐消失，角色视线向侧方偏移，紧张的氛围通过收紧的构图展现
```

### 示例3：冲突高潮

**原描述（不安全）：**
```
双方激烈打斗，拳脚相向
```

**安全替换：**
```
双方位置快速切换，残影效果，光影在两者之间快速交替，能量波动向外扩散，通过快速剪辑和色彩对比表现对抗强度
```

## 特殊场景处理

### 医疗/健康主题
- 使用示意图、图标、抽象动画
- 避免真实伤口、手术场景
- 使用绿色、蓝色等舒缓色调
- 强调治愈和康复

### 犯罪/悬疑主题
- 使用暗示性画面而非直接展示
- 通过对话和音效营造氛围
- 使用象征物（血迹符号、警示带等）
- 保持心理悬疑而非视觉冲击

### 灾难/事故主题
- 聚焦后果而非过程
- 使用数据可视化替代真实画面
- 强调预防和解决方案
- 使用希望和重建作为结尾

## 合规声明

所有生成的分镜脚本应：
1. 适合全年龄段观看
2. 不包含任何暴力或血腥暗示
3. 通过艺术化手法保持叙事张力
4. 符合各平台的社区规范

如有不确定内容，优先选择更保守的表达方式。

## ⏱️ 镜头时长规则

### 短视频优化（≤60秒）

| 总时长 | 建议镜头数 | 单镜头平均 | 开场镜头 |
|--------|-----------|-----------|---------|
| 30秒 | 3-5个 | 6-10秒 | 3-5秒 |
| 45秒 | 4-6个 | 7-11秒 | 4-5秒 |
| 60秒 | 5-8个 | 7-12秒 | 4-5秒 |

**短视频原则**：按镜头数量规划，避免过度拆分，单镜头不低于3秒

### 标准视频（>60秒）

| 时长 | 适用场景 | 节奏 |
|------|---------|------|
| <10秒 | 快节奏、开场、强调 | 紧凑 |
| 10-15秒 | 标准镜头（默认上限） | 适中 |
| 15-20秒 | 信息密集型（教学/数据） | 较慢 |
| >20秒 | 特殊叙事需求 | 很慢 |

**拆分策略：**
- 超过上限 → 拆分为2-3个子镜头
- 相邻镜头 → 必须有运镜变化
- 避免连续3个相同类型运镜

## 📐 构图法则速查

### 三分法
```
┌────┬────┬────┐
│ ★  │    │ ★  │ ← 上交叉点（地平线、眼睛）
├────┼────┼────┤
│    │主体│    │
├────┼────┼────┤
│ ★  │    │ ★  │ ← 下交叉点（CTA、重点）
└────┴────┴────┘
```

### 景别选择
- **开场**: 远景/全景 → 中景 → 特写
- **对话**: 中景/近景为主
- **情感**: 近景/特写
- **动作**: 全景/中景
- **结尾**: 特写 → 中景 → 远景

## 🎥 运镜情绪对照

| 运镜 | 速度 | 情绪 | 使用时机 |
|------|------|------|----------|
| 缓推 | 慢 | 沉思、紧张 | 内心戏 |
| 急推 | 瞬间 | 震惊、冲击 | 惊吓点 |
| 缓拉 | 慢 | 释然、结束 | 场景转换 |
| 揭示拉 | 中 | "原来如此" | 展现全貌 |
| 环绕 | 中 | 强调、360度 | 重要主体 |
| 上升 | 慢 | 超脱、渺小 | 结尾升华 |
| 手持 | 不定 | 真实、紧张 | 纪实/动作 |
| 变焦推 | 快 | 眩晕、压迫 | 心理冲击 |

## 🎭 内容安全红线

**禁止包含：**
- ❌ 暴力、血腥、恐怖
- ❌ 临时状态词（微笑、哭泣、愤怒）
- ❌ 令人不适的描述

**替换方案：**
- 微笑 → 嘴角自然弧度
- 哭泣 → 眼部湿润反光
- 愤怒 → 面部线条紧绷
- 奔跑 → 单脚离地身体前倾

## 基础运镜类型

### 推近 (Dolly In)
- **效果**：镜头向前推进，强调主体
- **参数**：spring dolly in（高系数，带轻微弹跳感）
- **适用场景**：开场、强调重要元素、展示细节
- **动画配合**：主体从底部spring升起并轻微旋转

### 拉远 (Dolly Out / Zoom Out)
- **效果**：镜头向后拉远，展示全景
- **参数**：dolly out with easeInOut, reverse dolly out
- **适用场景**：结尾、展示完整流程、过渡
- **动画配合**：元素汇聚中央后缓慢拉远

### 环绕 (Orbiting Shot)
- **效果**：镜头围绕主体360度旋转
- **参数**：360°半圈环绕，orbiting + tracking shot
- **适用场景**：展示3D结构、注意力机制、多维度信息
- **动画配合**：主体转向、连接线波纹扩散

### 跟拍 (Tracking Shot)
- **效果**：镜头跟随移动主体
- **参数**：smooth tracking shot
- **适用场景**：流程展示、序列流动、数据传递
- **动画配合**：输入序列逐层发光、流程箭头横贯全屏

### 摇移 (Pan)
- **效果**：镜头水平或垂直移动
- **参数**：pan right + dolly in, 轻摇强调层次
- **适用场景**：展示宽画面、过渡、强调层次
- **动画配合**：多列内容滚动、表格逐行高亮

### 特写 (Close-up)
- **效果**：近距离展示细节
- **参数**：close-up zoom
- **适用场景**：强调关键信息、生成结果展示
- **动画配合**：内容多行滚动、特效stagger闪烁

## 高级运镜组合

### 汇聚 (Converge)
- **效果**：所有元素快速汇聚到中央
- **参数**：spring converge
- **适用场景**：结尾过渡、强调主题
- **动画配合**：元素从四周stagger淡入

### 环绕+跟踪 (Orbiting + Tracking)
- **效果**：环绕主体同时跟随主体运动
- **参数**：orbiting + tracking shot
- **适用场景**：复杂3D结构、拟人化元素
- **动画配合**：拟人脑袋转头、连接线波纹扩散

## 运镜节奏控制

### 快速节奏
- spring() 高系数（stiffness: 100-150）
- 短暂持续时间（0.5-1秒）
- 配合stagger延迟

### 缓慢节奏
- easeInOut 缓动函数
- 较长持续时间（1-2秒）
- 配合interpolate多关键帧

### 弹跳节奏
- spring() 带阻尼（damping: 10-15）
- 轻微超调效果
- 适用于元素出现/消失

## 过渡效果

- **spring converge**：元素汇聚
- **stagger飞入**：从四周飞入
- **平移滑入**：从边缘滑入
- **淡入淡出**：透明度变化
- **缩放过渡**：元素放大/缩小

## 画面布局原则

1. **紧凑布局**：元素占屏80-90%
2. **多列设计**：grid/flex多列，减少留白
3. **分层展示**：嵌入/注意力/MLP三列网格
4. **动态填充**：元素环绕填充全屏

## 镜头序列组合示例

### 开场段落典型序列（3镜头）

```text
镜头1: 快速推近 (0-3秒)
   ┌────────┐           ┌────────┐           ┌────────┐
   │  帧A1  │    →      │  帧B1  │    →      │  帧C1  │
   │        │           │        │           │        │
   │        │           │  主角  │           │  主角  │
   │        │           │   ↑    │           │  ↑↑   │
   └────────┘           └────────┘           └────────┘
   全黑                 主角升起              标题出现

镜头2: 环绕展示 (3-8秒)
     ↗ ↑ ↖
   ↗  主角  ↖
  →  (360°)  ←
   ↘ 转动   ↙
     ↘ ↓ ↙

镜头3: 特写 (8-12秒)
   ┌────────┐
   │        │
   │   [眼] │ ← 特写面部细节
   │   睛   │
   └────────┘
```text

### 核心段落典型序列（2-4镜头）

```text
镜头1: 展示全景 (0-4秒)
   ┌─────────────────┐
   │   完整结构展示   │
   │                  │
   └─────────────────┘

镜头2: 分层拆解 (4-10秒)
   ┌─────────┬─────────┬─────────┐
   │  层1    │  层2    │  层3    │
   │  [ ]    │  [ ]    │  [ ]    │
   └─────────┴─────────┴─────────┘

镜头3: 流程跟踪 (10-15秒)
    → → → → → → →
    [输入]→[处理1]→[处理2]→[输出]

镜头4: 特写强调 (15-20秒)
   ┌──────┐
   │      │
   │ ⭐ 重点 └─ 特写高亮
   └──────┘
```text

### 结尾段落典型序列（2镜头）

```text

   ↗     ↖
  →  中央  ←
   ↘     ↙
     ↘ ↓ ↙
   元素从四周汇聚

镜头2: 拉远 (6-10秒)
   ┌────────┐           ┌────────────────┐
   │ [CTA]  │    →      │    [CTA]       │
   │  居中  │           │                │
   └────────┘           │ 全景展示        │
                        └────────────────┘
```text

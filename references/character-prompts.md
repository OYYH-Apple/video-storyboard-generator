# 角色提示词生成指南

## 何时需要角色提示词

对于以下视频类型，建议生成角色提示词：
- **剧情分镜** - 故事、短片、动画脚本
- **故事叙述** - 短片分镜、动画脚本、故事板
- **情感短片** - 品牌故事、公益广告

其他类型通常不需要角色提示词。

---

## 角色提示词结构

### 基础格式

```
[角色名称]: [主体描述] + [风格描述] + [质量和数量标签] + [比例和参数]
```

### 各部分说明

#### 1. 主体描述 (Primary Subject)

描述角色的基本特征：

**外貌特征：**
- 年龄（年轻、中年、老年）
- 性别（男性、女性、中性）
- 发型（短发、长发、卷发）
- 面部特征（眼镜、胡须、皱纹）

**衣着：**
- 衣服类型（衬衫、西装、休闲装）
- 颜色
- 配饰（帽子、眼镜、手表）

**表情/状态：**
- 表情（微笑、严肃、惊讶）
- 姿态（站立、坐着、行走）

#### 2. 风格描述 (Style)

**艺术风格：**
- 写实风格 (realistic)
- 卡通风格 (cartoon)
- 电影风格 (cinematic)
- 动画风格 (anime)
- 插画风格 (illustration)

**渲染风格：**
- 8k 分辨率 (8k)
- 4k 分辨率 (4k)
- 高质量 (high quality)
- 细节丰富 (highly detailed)
- 照片级 (photorealistic)

**光线：**
- 电影光效 (cinematic lighting)
- 自然光 (natural lighting)
- 柔和光线 (soft lighting)
- 戏剧性光线 (dramatic lighting)

#### 3. 质量和数量标签 (Quality and Quantity)

**质量标签：**
- 标准质量 (--q 1)
- 高质量 (--q 2)
- 超高质量 (--q 3)

**数量标签：**
- 1 张 (--ar 16:9)
- 4 张 (--ar 16:9 --n 4)
- 8 张 (--ar 16:9 --n 8)

#### 4. 比例和参数 (Aspect Ratio and Parameters)

**比例：**
- 宽横屏 16:9 --ar 16:9
- 纵向 9:16 --ar 9:16
- 方形 1:1 --ar 1:1
- 电影比例 2.35:1 --ar 21:9

**参数：**
- 风格化度 (--s 100-1000)
- 混乱度 (--c 0-100)
- 种子 (--seed 随机数)

## 生成角色提示词的工作流程

1. **收集角色信息**
   - 角色名称和定位（主角、配角、反派）
   - 基本特征（年龄、性别、外貌）
   - 衣着和配饰
   - 性格和情绪

2. **确定风格**
   - 根据视频类型选择艺术风格
   - 匹配整体视觉风格
   - 确定光线和氛围

3. **组合提示词**
   - 按照基础格式组合
   - 添加质量标签和参数
   - 确保提示词完整清晰

4. **优化和调整**
   - 根据AI生成结果调整描述
   - 添加必要的修饰符
   - 确保风格一致性

---

## 注意事项

1. **一致性**：同一角色的提示词应保持核心描述一致，只在场景、光线上变化
2. **简洁性**：避免过于复杂的描述，突出关键特征
3. **适配平台**：根据使用的AI绘图平台调整参数标签
4. **层次感**：通过光线和阴影营造层次感
5. **表情控制**：明确描述面部表情以传达情绪

## AI绘图平台适配

根据目标平台，提示词格式可能需要调整：

**Midjourney:**
```
[角色描述], [风格], --ar 16:9 --q 2
```

**Stable Diffusion:**
```
[角色描述], [风格], negative prompt: (worst quality, low quality:1.4), Steps: 20, CFG scale: 7, Sampler: Euler a
```

**DALL-E:**
```
A [角色描述] in [风格] style, high quality, detailed
```

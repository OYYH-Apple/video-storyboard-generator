## 一、基础构图法则

### 1.1 三分法构图 (Rule of Thirds)

将画面分为3×3网格，重要元素放置在交叉点或线上。

```text
┌────────┬────────┬────────┐
│   A    │        │   B    │  ← 上交叉点：适合放置标题、地平线
├────────┼────────┼────────┤
│        │  主体  │        │  ← 中心区域：主体位置
├────────┼────────┼────────┤
│   C    │        │   D    │  ← 下交叉点：适合放置CTA、关键信息
└────────┴────────┴────────┘
     A, B, C, D 为视觉焦点位置
```text

**适用场景：**
- 人物访谈：眼睛位于上1/3线
- 风景展示：地平线位于上1/3或下1/3线
- 产品展示：产品位于右下交叉点

### 1.2 黄金分割构图 (Golden Ratio)

使用1:1.618的比例关系确定主体位置，比三分法更自然和谐。

```text
黄金螺旋示意：
    ┌──────────────────┐
    │    ┌───────┐     │
    │    │ ┌───┐ │     │
    │    │ │ ◉ │ │  ← 主体位于螺旋中心
    │    │ └───┘ │     │
    │    └───────┘     │
    └──────────────────┘
```text

**使用技巧：**
- 将主要视觉焦点放置在螺旋收敛点
- 引导线沿螺旋方向延伸
- 适合需要优雅、艺术感的场景

### 1.3 对称构图 (Symmetry)

画面左右或上下完全对称，营造平衡、庄重、仪式感。

```text
对称构图示例：
┌──────────────────────────┐
│            │             │
│   左侧     │    右侧     │
│   镜像     │    镜像     │
│            │             │
│──────中心线──────────────│
│            │             │
└──────────────────────────┘
```text

**变体类型：**
- **垂直对称**：建筑、倒影
- **水平对称**：水面倒影、天地交界
- **放射对称**：旋转楼梯、圆形图案

### 1.4 框架式构图 (Framing)

利用前景元素构建"画中画"效果，增强层次感和纵深感。

```text
框架构图示意：
    ╔════════════════════╗
    ║  ┌────────────┐    ║
    ║  │            │    ║ ← 框架元素（门框、窗框、树枝等）
    ║  │   主体     │    ║
    ║  │   ◉        │    ║ ← 被框住的主体
    ║  │            │    ║
    ║  └────────────┘    ║
    ╚════════════════════╝
```text

**常用框架元素：**
- 建筑结构：门窗、走廊、拱门
- 自然元素：树枝、山洞、花朵
- 人工道具：相框、镜子、屏幕

## 二、电影级景别系统

### 2.1 远景 (Extreme Long Shot / ELS)

展示广阔环境，人物占画面比例极小（<10%）。

**视觉特征：**
- 环境 > 人物
- 强调空间感和氛围
- 人物与环境的关系

**使用时机：**
- 开场 establishing shot
- 展示宏大场景
- 表达孤独、渺小感

```text
远景构图：
┌─────────────────────────────┐
│  ~~~~~~~~~~~~~~~~~~~~~~~~~  │
│  ~    远山/城市/天空    ~  │
│  ~                       ~  │
│  ~         ●            ~  │ ← 人物仅为小点
│  ~        /|\           ~  │
│  ~~~~~~~~~~~~~~~~~~~~~~~~~  │
└─────────────────────────────┘
```text

### 2.2 全景 (Long Shot / LS)

展示人物全身和环境，人物占画面高度70-100%。

**视觉特征：**
- 人物全身可见
- 保留部分环境信息
- 展现人物动作和姿态

**使用时机：**
- 展示服装、造型
- 表现肢体语言
- 人物与环境互动

```text
全景构图：
┌────────────────────────┐
│      ┌────────┐        │
│      │  头部  │        │
│      │  躯干  │        │ ← 人物占画面大部分
│      │  腿部  │        │
│      └────────┘        │
│    地面/少量环境       │
└────────────────────────┘
```text

### 2.3 中景 (Medium Shot / MS)

展示人物膝盖以上，占画面高度50-70%。

**视觉特征：**
- 平衡人物与环境
- 适合对话场景
- 展现上半身手势

**使用时机：**
- 双人对话
- 产品演示
- 日常场景

```text
中景构图（膝盖以上）：
┌────────────────────────┐
│   ┌────────────────┐   │
│   │     头部       │   │
│   │  ┌──────────┐  │   │
│   │  │   躯干   │  │   │
│   │  │   手臂   │  │   │
│   └──┴──────────┴──┘   │
│        膝盖线          │
└────────────────────────┘
```text

### 2.4 近景 (Medium Close-Up / MCU)

展示人物胸部以上，占画面高度30-50%。

**视觉特征：**
- 强调面部表情
- 适合情感交流
- 建立亲密感

**使用时机：**
- 表达情绪
- 强调台词
- 建立人物关系

```text
近景构图（胸部以上）：
┌──────────────────┐
│   ┌──────────┐   │
│   │   头部   │   │
│   │  ◉    ◉  │   │ ← 面部细节清晰
│   │    ▽     │   │
│   ├──────────┤   │
│   │   肩膀   │   │
└───┴──────────┴───┘
```text

### 2.5 特写 (Close-Up / CU)

展示人物肩部以上或物体细节，占画面高度15-30%。

**视觉特征：**
- 高度聚焦
- 消除环境干扰
- 强调细节和情感

**使用时机：**
- 情感高潮
- 展示关键细节
- 创造视觉冲击

```text
特写构图（面部）：
┌────────────────┐
│ ┌────────────┐ │
│ │   额头     │ │
│ │ ◉      ◉   │ │ ← 眼睛细节
│ │     ▽      │ │ ← 鼻子
│ │   ═════    │ │ ← 嘴唇
│ └────────────┘ │
└────────────────┘
```text

### 2.6 大特写 (Extreme Close-Up / ECU)

展示极细节（眼睛、嘴唇、产品局部），占画面<15%。

**视觉特征：**
- 抽象化表达
- 强调质感
- 创造强烈视觉冲击

**使用时机：**
- 情感极致表达
- 产品质感展示
- 悬疑氛围营造

```text
大特写构图（单眼）：
┌────────────┐
│  ～～～～  │ ← 眉毛
│ ╭────────╮ │
│ │  ◉──◉  │ │ ← 眼睛细节
│ │   ◉◉   │ │ ← 瞳孔、反光
│ ╰────────╯ │
└────────────┘
```text

## 三、电影级画面布局方案

### 3.1 视觉重心布局 (Visual Weight)

根据元素重要性分配画面空间。

**布局原则：**
- 重要元素 > 大面积
- 次要元素 > 小面积
- 负空间 > 20-30%

```text
视觉重心布局示例：
┌────────────────────────────────┐
│                    ┌─────────┐ │
│     负空间         │  主角   │ │ ← 60%画面
│     (30%)          │  ◉      │ │
│                    │         │ │
│                    └─────────┘ │
│    ┌─────┐                     │
│    │次要 │                     │
│    └─────┘                     │
└────────────────────────────────┘
```text

### 3.2 引导线布局 (Leading Lines)

利用线条引导观众视线到主体。

**常用引导线：**
- 道路、走廊
- 栏杆、扶手
- 地平线、天际线
- 树枝、河流

```text
引导线布局：
        ↖ 视线方向
    ╱────────────────╲
   ╱    ～～～～～    ╲
  ╱    ～～～～～～    ╲ ← 汇聚线
 ╱    ～～～～～～～    ╲
╱         ●             ╲ ← 主体位置
    道路/走廊/栏杆
```text

### 3.3 层次布局 (Layering)

通过前景、中景、背景构建三维空间感。

```text
三层布局示意：
前景（虚化）   中景（清晰）   背景（虚化）
     │              │              │
   ╭───╮        ╭───────╮       ╭───╮
   │ 🌿 │   →   │  主角  │   →   │ 🏢 │
   ╰───╯        │   ◉   │       ╰───╯
                ╰───────╯
   增加深度      视觉焦点        环境信息
```text

**景深层次控制：**
- **浅景深**：f/1.4-f/2.8，背景虚化
- **中等景深**：f/5.6-f/8，整体清晰
- **深景深**：f/11-f/16，全部清晰

### 3.4 动态平衡布局 (Dynamic Balance)

通过不对称元素达到视觉平衡。

```text
动态平衡示例：
┌───────────────────────────────┐
│                               │
│   ┌───────────┐               │
│   │           │               │
│   │   大元素   │      小元素   │
│   │           │        ●      │ ← 小元素靠近边缘
│   └───────────┘               │    产生平衡
│                               │
└───────────────────────────────┘
```text

**平衡公式：**
- 大元素（靠近中心）× 小元素（靠近边缘）= 视觉平衡
- 深色元素 × 浅色元素 = 色彩平衡
- 复杂元素 × 简洁元素 = 信息平衡

## 四、电影色彩布局

### 4.1 色彩对比布局

利用色轮对比创造视觉冲击。

**对比类型：**
- **互补色**：红↔绿、蓝↔橙、黄↔紫
- **冷暖对比**：冷色调背景 + 暖色调主体
- **明暗对比**：深色背景 + 亮色主体

```text
色彩对比布局示例：
┌──────────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ 蓝色背景
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ (冷)
│ ▓▓▓▓▓▓▓▓ 🟠 ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ 橙色主体
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ (暖) 对比强烈
└──────────────────────────────┘
```text

### 4.2 色彩和谐布局

使用相邻色系创造和谐统一感。

**和谐类型：**
- **单色系**：同一色相的不同明度
- **类比色**：色轮相邻颜色（蓝→蓝绿→绿）
- **三色系**：色轮等距三色（红、黄、蓝）

```text
类比色布局示例：
深蓝 → 中蓝 → 浅蓝 → 蓝绿
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
  从左到右，色相平滑过渡
```text

## 五、画面留白艺术

### 5.1 呼吸空间 (Breathing Room)

为主体周围保留适当空白，避免画面拥挤。

**留白规则：**
- 人物朝向侧 > 2倍头部空间
- 运动方向前 > 3倍运动空间
- 视线方向前 > 视线落点空间

```text
视线方向留白：
┌────────────────────────────────┐
│     视线方向                    │
│  ════════════►                 │
│                                │
│    ┌───────┐                   │
│    │ ◉   ◉ │→→→→→→→ 空白区域  │
│    │   ▽   │    (2/3画面)      │
│    └───────┘                   │
│                                │
└────────────────────────────────┘
```text

### 5.2 负空间运用

利用空白区域传达情绪和氛围。

**负空间类型：**
- **极简主义**：大面积空白，强调主体
- **孤独感**：人物占据小比例，周围空旷
- **宏大感**：小主体 + 大空间

## 七、常见构图错误

### 7.1 需避免的问题

❌ **头部空间不足**
```text
头顶贴边：
┌────────────┐
│  ┌──────┐  │ ← 头顶太贴
│  │ 头   │  │
│  │身体  │  │
└──┴──────┴──┘
```text

✅ **正确做法**：头顶保留1/5画面空间

❌ **切关节**
```text
在膝盖/手腕处切分：
┌────────────┐
│  ┌──────┐  │
│  │ 上身 │  │
│  ├──────┤  │ ← 恰好在手腕
│  │      │  │
└──┴──────┴──┘
```text

✅ **正确做法**：在关节上方或下方切分

❌ **水平线居中**
```text
地平线居中：
┌────────────┐
│ ～～～～～ │
│━━━━━━━━━━━━│ ← 地平线居中
│            │
└────────────┘
```text

✅ **正确做法**：地平线在1/3或2/3处

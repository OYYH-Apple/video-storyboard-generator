# Seedance 2.0 提示词模板

Seedance 2.0 的提示词模板建议采用**结构清晰、分层描述**的方式，便于模型精准理解画面、动作、镜头、节奏、氛围以及台词/旁白。核心是充分利用多模态参考（图片、视频、音频），在提示词中明确说明“参考X号素材”。

## 推荐提示词模板

```markdown
[主体描述]：详细描述主要角色/物体外观、服装、年龄、体型、表情等（如果有参考图，写“严格遵循参考1号图片中的角色外观”）。

[场景与氛围]：描述环境、光线、天气、色调、整体风格（如电影级、写实、赛博朋克等）。

[动作与情节]：按时间顺序描述整个视频的完整流程，在需要说话的时刻标注“（角色说：XXX）”或“（旁白响起）”作为标记。动作要具体、可视化（如“双手缓缓上举，头发随风飘动，汗珠从额头滑落”）。

[台词与旁白]：精确写出每句台词/旁白的文字内容、说话角色、语气、语速、情感（如“低沉磁性男声，缓慢而坚定”）。如果有参考音频，写“台词1严格使用参考音频1的语音和语调”。

[镜头语言]：明确镜头运动和剪辑方式（如“从低机位缓慢推近”“360度环绕拍摄”“快速切镜”“长镜头跟随”）。

[音画同步/音频]：描述背景音乐、音效、节奏，以及台词/旁白与画面的同步要求（如“台词在角色张嘴时精确对齐唇动”“旁白在镜头推近时渐强”）。如果有参考音频，说明如何同步。

[其他控制]：时长（建议15秒以内）、分辨率（1080p或更高）、质量要求（如“高度逼真，细节丰富，无畸变”）。
```

## 为什么这个模板有效？
- 层次分明，避免信息混杂导致模型生成畸变。
- 动作与台词分离，便于模型分别处理视觉和音频。
- 明确标注参考素材和同步要求，能显著提升一致性、唇同步和可用率。

## 示例1：舞蹈视频（结合多模态参考）

**上传素材**：1张女舞者正面照（参考图1）、1段参考舞蹈视频片段（参考视频1）、1段背景音乐（参考音频1）

**提示词**：
```
[主体描述]：一位20岁亚洲女性舞者，严格遵循参考1号图片中的面容、体型、长直发、穿着白色紧身舞衣，表情自信而专注。

[场景与氛围]：现代舞蹈室，柔和顶光+侧光，木地板反射光泽，背景干净，整体电影级写实风格，温暖色调。

[动作与情节]：0-3秒：舞者从静止开始，双手缓缓举过头顶，身体微微后仰，头发轻甩；3-8秒：快速旋转三圈，裙摆飞扬，脚尖点地；8-15秒：高难度托举动作后落地，单膝跪地，抬头直视镜头，汗珠滑落，结束pose。

[台词与旁白]：无

[镜头语言]：开头用中景缓慢推近到特写脸部，旋转时360度环绕镜头，跟随动作流畅移动，最后拉远全景。

[音画同步/音频]：动作节奏严格与参考音频1的鼓点同步，旋转时鼓点加快，落地时音乐高潮，加入轻微呼吸声和脚步声。

[其他控制]：15秒，1080p，高度逼真，布料和头发物理动态自然。
```

## 示例2：产品广告短片（带旁白）

**上传素材**：1张手机产品图（参考图1）

**提示词**：
```
[主体描述]：一款银色未来感智能手机，严格遵循参考1号图片中的外观和细节，手持在画面中。

[场景与氛围]：夜晚城市天台，霓虹灯闪烁，雨后湿润地面反射灯光，赛博朋克风格，冷蓝色调。

[动作与情节]：0-4秒：男士手持手机从口袋取出，屏幕亮起显示炫酷界面；4-10秒：手机悬浮在空中缓慢旋转360度，展示机身细节，水珠从屏幕滑落；10-15秒：手机飞向镜头，屏幕显示品牌LOGO，爆炸式粒子特效，旁白响起。

[台词与旁白]：旁白（第4-15秒）：“全新一代智能手机，重新定义未来。更快、更强、更智能。” 语气：激昂自信的男声，语速稍快，带有科技感回响。

[镜头语言]：开头低机位仰拍手部动作，中段环绕镜头展示手机，结尾快速推镜到屏幕特写。

[音画同步/音频]：旁白从第4秒开始渐入，与手机旋转节奏同步，每句话对应一个展示镜头，结尾高潮与粒子特效同时爆发，背景电子音乐节奏感强。

[其他控制]：15秒，电影级渲染，细节丰富，光影真实，无畸变。
```

## 示例3：剧情短片（带角色台词和旁白）

**上传素材**（可选）：1段男声旁白音频（参考音频1）、1段背景音乐（参考音频2）

**提示词**：
```
[主体描述]：一位30岁亚洲男性，短发，穿着黑色西装，严肃表情，站在雨中。

[场景与氛围]：夜晚都市街头，大雨倾盆，路灯反射在水洼中，冷蓝色调，电影级写实风格。

[动作与情节]：0-4秒：男人站在街头，低头点烟，雨水打在脸上；4-8秒：抬头看向远处，眼神坚定，开始说话（说台词1）；8-12秒：缓缓向前走，雨伞被风吹翻；12-15秒：停下脚步，握紧拳头，旁白响起。

[台词与旁白]：
- 角色台词1（第4-8秒）：“我不会放弃，无论代价多大。” 语气：低沉有力，略带沙哑，语速中等，男性声音。
- 旁白（第12-15秒）：“在风雨中，他选择了坚持。” 语气：深沉磁性男声，缓慢而富有哲理。如果有参考音频1，使用参考音频1的语调和节奏。

[镜头语言]：开头全景展示雨夜环境，4秒时推近到脸部特写（说话时），12秒时缓慢拉远。

[音画同步/音频]：背景雨声持续，台词1时唇形精确同步说话动作，旁白与背景音乐参考音频2的高潮部分对齐，声音渐强。整体音效真实，雨滴声与脚步同步。

[其他控制]：15秒，1080p，高度逼真，雨水物理效果自然。
```

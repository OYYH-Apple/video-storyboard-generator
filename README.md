# Video Storyboard Generator

> 🎬 **Claude Skill** - 通用视频分镜脚本生成器，支持头脑风暴、用户偏好记忆、15秒镜头规则、角色提示词生成、音效/BGM建议、极简版/专业版双模式。

---

## 🎯 核心特性

### 1. 全类型视频支持

支持 **7大类14种** 视频类型：

| 分类 | 类型 | 适用场景 |
|------|------|----------|
| 概念讲解 | 技术教程、概念科普 | AI原理、区块链机制、物理定律 |
| 书籍介绍 | 整书概览、章节导读 | 书籍推荐、章节解读、读书笔记 |
| 热点评论 | 事件分析、观点表达 | 新闻解读、事件分析、观点表达 |
| 剧情分镜 | 故事叙述、情感短片 | 短片分镜、动画脚本、品牌故事 |
| 产品介绍 | 产品推广、功能演示 | 产品发布、功能演示、App介绍 |
| 数据报告 | 数据洞察、对比分析 | 数据可视化、研究报告、趋势分析 |
| 教程指南 | 操作指南 | 软件教程、流程说明、使用指南 |

### 2. 用户画像个性化

系统会记忆你的偏好，随着使用次数增加提供越来越精准的推荐：

- 视觉风格偏好
- 主角/吉祥物偏好
- 旁白风格偏好
- 运镜节奏偏好

### 3. 15秒黄金法则

**核心规则：单个镜头时长不超过15秒**

超过15秒的镜头自动拆分为多个子镜头，确保：
- 视觉节奏紧凑，保持观众注意力
- 每个镜头聚焦单一视觉焦点
- 镜头间有明确的运镜变化

### 4. 多镜头序列设计

每个段落包含多个独立镜头，精确控制视频节奏：

### 5. 角色提示词生成（故事/剧情类）

为故事类、电影类视频自动生成 AI 绘画提示词：

**包含内容：**
- 角色外貌特征（年龄、性别、衣着）
- 风格描述（写实、卡通、电影风格）
- 质量和参数标签（8k、4k、比例）
- 支持 Midjourney、Stable Diffusion、DALL-E

**适用场景：**
- 剧情分镜、故事叙述
- 短片分镜、动画脚本
- 品牌故事、情感短片

### 6. 音效/BGM 建议

为每个镜头提供配套的音效和背景音乐建议：

**音效类型：**
- 环境音（雨声、风声、城市噪音）
- 转场音效（swoosh、glitch、fade）
- 强调音效（click、ding、pop）

**BGM 建议：**
- 音乐类型（电子、古典、流行、氛围）
- 节奏和情绪（紧张、轻松、激昂）
- 音量建议（根据内容类型调整）

### 7. 双模式输出

**极简版：**
- 仅包含画面内容和时间描述
- 简洁易读，快速规划
- 适合初步构思和头脑风暴

**专业版：**
- 详细运镜指导（运镜类型、速度、方向）
- 画面布局规范（元素位置、占比）
- 视觉元素说明（具体元素描述）
- 过渡效果（镜头间过渡）
- 音效/BGM 建议（可选）
- 角色提示词（故事/剧情类）
- OpenCode 执行提示词（可直接复制）

```markdown
段落1：开场引入（0-12秒）

**镜头序列**（共3个镜头，每个≤15秒）：
#### 镜头1：推近 (Dolly In)（0-3秒）
- **运镜**：从全黑快速推近
- **布局**：主角占屏 40%
- **视觉**：标题文字飞入
- **过渡**：无缝过渡

#### 镜头2：环绕 (Orbiting)（3-8秒）
- **运镜**：360°环绕展示
- **布局**：主角居中
- **视觉**：粒子动态流动
- **过渡**：平滑过渡

#### 镜头3：特写 (Close-up)（8-12秒）
- **运镜**：推近到关键信息
- **布局**：关键词气泡
- **视觉**：气泡逐个出现
- **过渡**：平滑过渡

**旁白**：开场白内容
```

## 🚀 如何使用

**直接在 Claude 对话中使用：**

```
我想做一个[你的视频主题]的分镜脚本
```

**Claude 会执行 7 阶段工作流程：**

1. 🧠 **加载用户画像** - 读取你的历史偏好
2. 🎯 **视频类型识别** - 自动识别视频类型
3. 💬 **头脑风暴引导** - 针对性澄清需求
4. 📋 **段落结构设计** - 根据类型定制段落
5. ✂️ **镜头拆分** - 15秒规则拆分
6. 🎬 **生成分镜脚本** - 生成完整脚本
7. 🔄 **更新用户画像** - 记录本次偏好

**示例对话：**

```
用户：我想做一个《人类简史》的书籍介绍视频

Claude：我注意到你之前偏好文艺温暖风格，这次继续使用吗？

用户：是的

Claude：好的，让我确认几个问题：
- 是整书概览还是单章节导读？
- 目标读者是谁？
- 视频时长预期多久？
- 需要突出哪些核心观点？
...
```

## ⚙️ 配置文件

### 默认配置 (`config/default-config.yaml`)

```yaml
video:
  fps: 30
  default_duration: 60

visual:
  background_style: "深蓝渐变 + 神经网络线条流动 + 微光粒子"
  main_color: "蓝色"

output:
  directory: "./docs"
```

### 视频类型配置 (`config/video-types.yaml`)

每种类型包含：
- `visual_style` - 视觉风格
- `character` - 主角设定
- `narration_style` - 旁白风格
- `camera_rhythm` - 运镜节奏
- `max_shot_duration` - 最大镜头时长

## 🎬 15秒拆分规则

### 拆分策略

```
原镜头时长        拆分建议
15秒以内         保持单个镜头
16-30秒          拆分为2个镜头
31-45秒          拆分为3个镜头
45秒以上         拆分为N个镜头
```

### 运镜变化规则

拆分后的相邻镜头必须有运镜变化：

| 前一个镜头 | 推荐后一个镜头 |
|-----------|---------------|
| 推近 | 环绕/特写/拉远 |
| 环绕 | 推近/跟拍/摇移 |
| 跟拍 | 推近/特写/定格 |
| 拉远 | 推近/汇聚 |
| 特写 | 拉远/环绕/摇移 |
